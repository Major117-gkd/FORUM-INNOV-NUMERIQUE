# Forum de l'Innovation Numérique - Université de Labé 🚀

Bienvenue sur le dépôt officiel de la plateforme web du **Forum de l'Innovation Numérique et de l'Entrepreneuriat Étudiant** (FINEE) de l'Université de Labé, édition 2026.

![Aperçu du Projet](https://images.unsplash.com/photo-1581091226825-a6a2a5aee158?ixlib=rb-1.2.1&auto=format&fit=crop&w=1350&q=80)

## 📋 À Propos du Projet

Cette plateforme a été conçue pour promouvoir l'innovation technologique et l'esprit entrepreneurial au sein de l'Université de Labé. Elle sert de vitrine pour l'événement, permettant aux participants de découvrir le programme, de connaître les intervenants et de s'inscrire aux différentes sessions.

**Slogan :** *« Innove. Entreprends. Connecte-toi. »*

## ✨ Fonctionnalités Clés

- 🏠 **Page d'Accueil Dynamique** : Présentation des objectifs du forum et des chiffres clés.
- 📅 **Programme Complet** : Détails des conférences, ateliers et panels sur plusieurs jours.
- 👤 **Intervenants Vedettes** : Mise en avant des experts et leaders d'opinion invités.
- 📝 **Système d'Inscription** : Formulaire de participation intuitif et validé.
- 📱 **Design Responsive** : Optimisé pour tous les écrans (Mobiles, Tablettes, Ordinateurs).
- 🎨 **Esthétique Premium** : Utilisation de Glassmorphism, animations fluides au défilement et typographie moderne.

## 🛠️ Technologies Utilisées

- **Structure :** HTML5 Sémantique
- **Style :** CSS3 (Variables CSS, Flexbox, CSS Grid, Glassmorphism)
- **Logique :** JavaScript (Vanilla JS, Animations de révélation, Navigation collante)
- **Design :** Polices Google (Poppins/Outfit), Icônes personnalisées en SVG

## 📁 Structure du Projet

```text
/RCUN-3
├── assets/             # Images, icônes et favicons
├── index.html          # Page d'accueil principale
├── inscription.html    # Page du formulaire d'inscription
├── programme.html      # Page de l'agenda de l'événement
├── main.js             # Scripts d'animation et d'interaction
└── style.css           # Système de design global et styles
```

## 🚀 Installation Locale

Pour visualiser le projet sur votre machine locale :

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/[votre-nom-utilisateur]/RCUN-3.git
   ```
2. Ouvrez le fichier `index.html` dans votre navigateur préféré.
3. *Alternative :* Utilisez l'extension **Live Server** sur VS Code pour un rendu en temps réel.
